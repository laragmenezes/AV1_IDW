# Lista de Tarefas

Uma aplicação web simples para gerenciar uma lista de tarefas. Permite adicionar, editar e excluir tarefas de forma intuitiva.

## Funcionalidades

- **Adicionar tarefa**: Digite uma tarefa no campo de entrada e clique em "Adicionar".
- **Editar tarefa**: Clique no botão "Editar" ao lado da tarefa para modificá-la.
- **Excluir tarefa**: Clique no botão "Excluir" para remover a tarefa da lista.
- **Validação**: Impede a adição de tarefas vazias.

## Como usar

1. Abra o arquivo `index.html` em um navegador web moderno.
2. Digite uma tarefa no campo de entrada.
3. Clique em "Adicionar" para incluir na lista.
4. Use os botões "Editar" ou "Excluir" conforme necessário.

## Tecnologias utilizadas

- **HTML**: Estrutura da página.
- **CSS**: Estilização com tema rosa.
- **JavaScript**: Lógica para manipulação das tarefas.
